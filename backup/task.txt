Задача писать бекап:

На вход по ключу файл конфигурации.

В файле конфигурации содержатся 
1) набор папок/файлов 
2) дериктория в которую нужно сохранить бекап 
3) максимальное количество файлов в дериктории бекапа

Задача зарахивировать весь набор файлов в один файл и положить его в указанную дерикторию

Если в указанной дериктории количество файлов равно максимальному необходимо удалить самый старый бекап

Файл конфигурации в любом удобном формате
